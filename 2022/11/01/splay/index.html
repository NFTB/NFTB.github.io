Splay

1. 基础知识

1.1 简介

Splay 是一种二叉查找树，它通过不断将某个节点旋转到根节点，使得整棵树仍然满足二叉查找树的性质，并且保持平衡而不至于退化为链。它由 Daniel Sleator 和 Robert Tarjan 发明
Splay通过不断旋转保证树的平衡，在不断旋转中Splay的中序遍历不发生变化
Splay可以完成许多线段树难以完成的操作
模板级Splay一般维护的变量如下

展开代码 


struct node{
    int s[2],fa,val;
    s[0]是左儿子，s[1]是右儿子
    val是当前节点的值,fa是父节点
    int size,flag;
    size是包括当前节点和所有子节点的中节点个数
    flag便是懒标记
}tr[N];

1.2 旋转操作

Splay的关键操作，分为左旋和右旋。
下图便是左旋的大致样子，也是我敲模板时想的例子。

可以看到共有四个节点发生变化
即x的父节点变为z,左儿子变为y；y的父节点变为x,右儿子变为b；b的父节点变为y，z的左儿子变为x
右旋就是把左旋反过来
集前人之智慧要分四种情况的旋转操作仅需一个函数便能完成

展开代码 


void rotate(int x){
    int y=tr[x].fa,z=tr[y].fa;
    int k=tr[y].s[1]==x;//确定三个点的左右关系
    tr[z].s[tr[z].s[1]==y]=x,tr[x].fa=z;//更新z和x
    tr[y].s[k]=tr[x].s[k^1],tr[tr[x].s[k^1]].fa=y;//更新y和b
    tr[x].s[k^1]=y,tr[y].fa=x;//更新x和y
    pushup(y),pushup(x);
}

1.3 Splay操作

Splay规定：每访问一个节点后都要强制将其旋转到根节点
Splay操作便是将一个点(x)旋转到另一个节点(k)下面
Splay一共有六种操作

可见当父节点已经是根(k)的时候只需再旋转一次其他情况要旋转两次
可以看到当xyz三者是直链时才需要对y进行旋转便可以得到如下代码

展开代码 


void splay(int x,int k){
    while(tr[x].fa!=k){
        int y=tr[x].fa,z=tr[y].fa;
        if(z!=k)//父节点不是根(k)
            if((tr[z].s[1]==y)^(tr[y].s[1]==x)) rotate(x);
            else rotate(y);//是直链旋转y
        rotate(x);
    }
    if(!k) root=x;//旋转至根要更新根
}

1.4 插入操作

插入操作比较简单，我们通过中旬遍历维护的val来确定新点的位置
这里直接给出代码

展开代码 


void insert(int x){
    int u=root,p=0;
    while(u) p=u,u=tr[u].s[tr[u].val<x];//判断在左儿子还是右儿子
    u=++idx;//分配节点
    if(p) tr[p].s[tr[p].val<x]=u;//非根节点修改父节点与该节点的关系
    tr[u].fa=p,tr[u].val=x;
    splay(u,0);//Splay规定操作保证复杂度
}

1.5 查询中序遍历排名k的数

有了上述的两种操作如果我们相对区间l,r进行操作
可以先Splay(l-1,0),Splay(r+1,l-1)将需进行操作的区间都整到r节点的左儿子方便进行删除修改等操作。
所以当给出区间l,r时我们需要得到l-1和r+1这两个点的值
我们维护了size，便可以通过size不断递归找到排名k的数

展开代码 


int get_k(int k){
    int u=root;
    while(true){//寻找时一定能找到才调用该函数
        pushdown(u);
        if(tr[tr[u].s[0]].size>=k) u=tr[u].s[0];//在左边
        else if(tr[tr[u].s[0]].size+1==k) return u;
        else k-=tr[tr[u].s[0]].size+1,u=tr[u].s[1];//在右边
    }
    return -1;
}

2. 总结

有了基础操作，通过pushup,pushdown灵活变通便可以解决各种问题
Splay属于代码量适中能完成线段树、treap难以完成操作的数据结构
下面给出NOI的一道题目介绍一下Splay的其他各种操作

[NOI2005] 维护数列

题目描述

请写一个程序，要求维护一个数列，支持以下 $6$ 种操作：

编号 名称 格式 说明
1 插入 $\operatorname{INSERT}\ posi \ tot \ c_1 \ c_2 \cdots c_{tot}$ 在当前数列的第 $posi$ 个数字后插入 $tot$ 个数字：$c_1, c_2 \cdots c_{tot}$；若在数列首插入，则 $posi$ 为 $0$
2 删除 $\operatorname{DELETE} \ posi \ tot$ 从当前数列的第 $posi$ 个数字开始连续删除 $tot$ 个数字
3 修改 $\operatorname{MAKE-SAME} \ posi \ tot \ c$ 从当前数列的第 $posi$ 个数字开始的连续 $tot$ 个数字统一修改为 $c$
4 翻转 $\operatorname{REVERSE} \ posi \ tot$ 取出从当前数列的第 $posi$ 个数字开始的 $tot$ 个数字，翻转后放入原来的位置
5 求和 $\operatorname{GET-SUM} \ posi \ tot$ 计算从当前数列的第 $posi$ 个数字开始的 $tot$ 个数字的和并输出
6 求最大子列和 $\operatorname{MAX-SUM}$ 求出当前数列中和最大的一段子列，并输出最大和

输入格式

第一行包含两个整数 $N$ 和 $M$，$N$ 表示初始时数列中数的个数，$M$ 表示要进行的操作数目。

第二行包含 $N$ 个数字，描述初始时的数列。以下 $M$ 行，每行一条命令，格式参见问题描述中的表格。

输出格式

对于输入数据中的 $\operatorname{GET-SUM}$ 和 $\operatorname{MAX-SUM}$ 操作，向输出文件依次打印结果，每个答案（数字）占一行。

样例 #1

样例输入 #1
9 8 
2 -6 3 5 1 -5 -3 6 3 
GET-SUM 5 4
MAX-SUM
INSERT 8 3 -5 7 2
DELETE 12 1
MAKE-SAME 3 3 2
REVERSE 3 6
GET-SUM 5 4
MAX-SUM
样例输出 #1
-1
10
1
10

提示

数据规模与约定
  • 你可以认为在任何时刻，数列中至少有 $1$ 个数。
  • 输入数据一定是正确的，即指定位置的数在数列中一定存在。
  • 对于 $50%$ 的数据，任何时刻数列中最多含有 $3 \times 10^4$ 个数。
  • 对于 $100%$ 的数据，任何时刻数列中最多含有 $5 \times 10^5$ 个数，任何时刻数列中任何一个数字均在 $[-10^3, 10^3]$ 内，$1 \le M \le 2 \times 10^4$，插入的数字总数不超过 $4 \times 10^6$。

（1）所需维护的数据

后两个是经典的线段树操作，维护sum（区间和）,ls（左连续字段和）,rs（右连续字段和）,ms（最大字段和）即可
reverse操作和same操作用两个懒标记维护

struct node{
    int s[2],fa,val;
    int rev,same;
    int size,ms,ls,rs,sum;
    void init(int _val,int _fa){
        fa=_fa,val=_val;
        rev=same=0;
        s[0]=s[1]=0;
        size=1,sum=ms=val;
        ls=rs=max(val,0);
    }
}tr[N];

（2）insert操作

在本题中一次插入一段数据，对区间进行操作如上所述
一段数据进行插入可以先对这段数据构建一个平衡树然后接在r节点的左边。

int built(int p,int l,int r){
    int mid=l+r>>1;
    int u=nodes[tt--];
    tr[u].init(w[mid],p);
    if(l<mid) tr[u].s[0]=built(u,l,mid-1);
    if(mid<r) tr[u].s[1]=built(u,mid+1,r);
    pushup(u);
    return u;
}

（3）delete操作

和insert操作一样，后面将r的左儿子置为0
这里主要介绍一下回收删除节点的操作

//对比builit的nodes[tt--]这里进行了回收
void dfs(int u){
    if(tr[u].s[0]) dfs(tr[u].s[0]);
    if(tr[u].s[1]) dfs(tr[u].s[1]);
    nodes[++tt]=u;//回收节点
}

(7) 完整代码

pushup和pushdown操作就是一些维护操作，做过线段树最大字段和的话很容易写

展开代码 


#include<bits/stdc++.h>
using namespace std;

const int N=5e5+5,INF=1e9;
int n,m;
int root,nodes[N],tt,w[N];
struct node{
    int s[2],fa,val;
    int rev,same;
    int size,ms,ls,rs,sum;
    void init(int _val,int _fa){
        fa=_fa,val=_val;
        rev=same=0;
        s[0]=s[1]=0;
        size=1,sum=ms=val;
        ls=rs=max(val,0);
    }
}tr[N];
void pushup(int u){
    int l=tr[u].s[0],r=tr[u].s[1];
    tr[u].size=tr[l].size+tr[r].size+1;
    tr[u].sum=tr[l].sum+tr[r].sum+tr[u].val;
    tr[u].ms=max({tr[l].ms,tr[r].ms,tr[l].rs+tr[r].ls+tr[u].val});
    tr[u].ls=max(tr[l].ls,tr[l].sum+tr[u].val+tr[r].ls);
    tr[u].rs=max(tr[r].rs,tr[l].rs+tr[u].val+tr[r].sum);
}
void pushdown(int u){
    int l=tr[u].s[0],r=tr[u].s[1];
    if(tr[u].same){
        tr[u].same=tr[u].rev=0;
        if(tr[u].s[0]) tr[l].same=1,tr[l].val=tr[u].val,tr[l].sum=tr[l].size*tr[l].val;
        if(tr[u].s[1]) tr[r].same=1,tr[r].val=tr[u].val,tr[r].sum=tr[r].size*tr[r].val;
        if(tr[u].val>0){
            if(tr[u].s[0]) tr[l].ls=tr[l].rs=tr[l].ms=tr[l].sum;
            if(tr[u].s[1]) tr[r].ls=tr[r].rs=tr[r].ms=tr[r].sum;
        }else{
            if(tr[u].s[0]) tr[l].ls=tr[l].rs=0,tr[l].ms=tr[l].val;
            if(tr[u].s[1]) tr[r].ls=tr[r].rs=0,tr[r].ms=tr[r].val;
        }
    }else if(tr[u].rev){
        tr[u].rev=0,tr[l].rev^=1,tr[r].rev^=1;
        swap(tr[l].ls,tr[l].rs),swap(tr[r].ls,tr[r].rs);
        swap(tr[l].s[0],tr[l].s[1]),swap(tr[r].s[1],tr[r].s[0]);
    }
}
void rotate(int x){
    int y=tr[x].fa,z=tr[y].fa;
    int k=tr[y].s[1]==x;
    tr[z].s[tr[z].s[1]==y]=x,tr[x].fa=z;
    tr[y].s[k]=tr[x].s[k^1],tr[tr[x].s[k^1]].fa=y;
    tr[x].s[k^1]=y,tr[y].fa=x;
    pushup(y),pushup(x);
}
void splay(int x,int k){
    while(tr[x].fa!=k){
        int y=tr[x].fa,z=tr[y].fa;
        if(z!=k)
            if((tr[z].s[1]==y)^(tr[y].s[1]==x)) rotate(x);
            else rotate(y);
        rotate(x);
    }
    if(!k) root=x;
}
int get_k(int k){
    int u=root;
    while(true){
        pushdown(u);
        if(tr[tr[u].s[0]].size>=k) u=tr[u].s[0];
        else if(tr[tr[u].s[0]].size+1==k) return u;
        else k-=tr[tr[u].s[0]].size+1,u=tr[u].s[1];
    }
    return -1;
}
void dfs(int u){
    if(tr[u].s[0]) dfs(tr[u].s[0]);
    if(tr[u].s[1]) dfs(tr[u].s[1]);
    nodes[++tt]=u;
}
int built(int p,int l,int r){
    int mid=l+r>>1;
    int u=nodes[tt--];
    tr[u].init(w[mid],p);
    if(l<mid) tr[u].s[0]=built(u,l,mid-1);
    if(mid<r) tr[u].s[1]=built(u,mid+1,r);
    pushup(u);
    return u;
}
void slove(){
    for(int i=1;i<N;i++) nodes[++tt]=i;
    cin>>n>>m;
    tr[0].ms=w[0]=w[n+1]=-INF;
    for(int i=1;i<=n;i++) cin>>w[i];
    root=built(0,0,n+1);
    string s;
    while(m--){
        cin>>s;
        if(s=="INSERT"){
            int posi,tot;
            cin>>posi>>tot;
            for(int i=0;i<tot;i++) cin>>w[i];
            int l=get_k(posi+1),r=get_k(posi+2);
            splay(l,0),splay(r,l);
            int u=built(r,0,tot-1);
            tr[r].s[0]=u;
            pushup(r),pushup(l);
        }else if(s=="DELETE"){
            int posi,tot;
            cin>>posi>>tot;
            int l=get_k(posi),r=get_k(posi+tot+1);
            splay(l,0);splay(r,l);
            dfs(tr[r].s[0]);
            tr[r].s[0]=0;
            pushup(r),pushup(l);
        }else if(s=="MAKE-SAME"){
            int posi,tot,c;
            cin>>posi>>tot>>c;
            int l=get_k(posi),r=get_k(posi+tot+1);
            splay(l,0);splay(r,l);
            auto &son=tr[tr[r].s[0]];
            son.same=1,son.val=c,son.sum=c*son.size;
            if(c>0) son.ls=son.rs=son.ms=son.sum;
            else son.ls=son.rs=0,son.ms=c;
            pushup(r),pushup(l);
        }else if(s=="REVERSE"){
            int posi,tot;
            cin>>posi>>tot;
            int l=get_k(posi),r=get_k(posi+tot+1);
            splay(l,0);splay(r,l);
            auto &son=tr[tr[r].s[0]];
            son.rev^=1;
            swap(son.ls,son.rs);
            swap(son.s[0],son.s[1]);
            pushup(r),pushup(l);
        }else if(s=="GET-SUM"){
            int posi,tot;
            cin>>posi>>tot;
            int l=get_k(posi),r=get_k(posi+tot+1);
            splay(l,0);splay(r,l);
            cout<<tr[tr[r].s[0]].sum<<'\n';
        }else cout<<tr[root].ms<<'\n';
    }
}
int main(){
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);std::cout.tie(nullptr);
    int _=1;
    // cin>>_;
    for(int i=1;i<=_;i++){
        slove();
    }
    return 0;
}
