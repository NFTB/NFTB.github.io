莫比乌斯反演

1.莫比乌斯函数

1.1算数基本定理：

任何一个大于1的正整数都能分解为有限个质数的乘积
$$
N={p_1}^{c_1}{p_2}^{c_2}\ldots{p_k}^{c_k}
$$
其中$c_i$都是正整数,$p_i$都是质数，且满足$p_1<p_2<\ldots<p_k$

1.3莫比乌斯函数的性质

证明：
当$n=1$时显然
当$n>1$时
由于$d$是$n$的约数，所以$d$可以由$n$的质因子组成
$n={p_1}^{c_1}{p_2}^{c_2}\ldots{p_k}^{c_k}$
$\exists c_i>1$时$\mu(n)=0$
故$d={p_1}^{\alpha_1}{p_2}^{\alpha_2}\ldots{p_k}^{\alpha_k}$其中$\alpha_i$等于$0$或$1$
所以
$$
\sum_{d|n}{\mu(d)}=C_{k}^{i}(-1)^i=(1+(-1))^k=0
$$

2.莫比乌斯反演

2.1结论

设$F(n)$,$f(n)$为两个数论函数
(1) 如果有
$$F(n)=\sum_{d|n}{f(d)}$$
那么有
$$f(n)=\sum_{d|n}\mu(d)F(\frac{n}{d})$$
(2) 如果有
$$F(n)=\sum_{n|d}{f(d)}$$
那么有
$$f(n)=\sum_{n|d}\mu(\frac{d}{n})F(d)$$

2.2证明

(1) 证明：
$$
\sum_{d|n}\mu(d)F(\frac{n}{d})
=\sum_{d|n}\mu(d)\sum_{i|\frac{n}{d}}{f(i)}
$$
这里交换求和次序
$$
d|n,i|\frac{n}{d}\Rightarrow i|n,d|\frac{n}{i}
$$
故
$$
\sum_{d|n}\mu(d)\sum_{i|\frac{n}{d}}{f(i)}=\sum_{i|n}f(i)\sum_{d|\frac{n}{i}}\mu(d)
$$
由莫比乌斯函数的性质可知当$\frac{n}{i}=1$即$n=i$时$$\sum_{d|\frac{n}{i}}\mu(d)=1$$
故
$$
f(n)=\sum_{d|n}\mu(d)F(\frac{n}{d})
$$
(2) 证明：
$$
\sum_{n|d}\mu(\frac{d}{n})F(d)=\sum_{n|d}\mu(\frac{d}{n})\sum_{d|i}f(i)
$$
同样交换积分次序
$$
n|d,d|i\Rightarrow n|i,\frac{d}{n}|\frac{i}{n}
$$
故
$$
\sum_{n|d}\mu(\frac{d}{n})\sum_{d|i}f(i)=\sum_{n|i}f(i)\sum_{\frac{d}{n}|\frac{i}{n}}\mu(\frac{d}{n})\overset{d^{‘}=\frac{d}{n}}{=}\sum_{n|i}f(i)\sum_{d^{‘}|\frac{i}{n}}\mu(d^{‘})
$$
同样的由莫比乌斯函数的性质可知当$\frac{n}{i}=1$即$n=i$时$$\sum_{d^{‘}|\frac{n}{i}}\mu(d^{‘})=1$$
故
$$f(n)=\sum_{n|d}\mu(\frac{d}{n})F(d)$$

3.数论分块

在举例应用之前，先介绍一下数论分块，它与莫比乌斯反演应用密不可分，可以在$O(\sqrt{n})$的时间内计算$\sum_{i=1}^{n}\lfloor\frac{n}{i}\rfloor$

3.1引理1

$$
\forall a,b,c\in \mathbb{Z},\lfloor \frac{a}{bc} \rfloor=\lfloor \frac{\lfloor \frac{a}{b} \rfloor}{c} \rfloor
$$
上式可以通过下式简单证明
$$
\frac{a}{b}=\lfloor \frac{a}{b} \rfloor +r (0\leq r<1)
$$

3.2引理2

$$
\forall n\in \mathbb{N_+},|{\lfloor \frac{n}{d} \rfloor |d \in \mathbb{N_+},d \leq n }|\leq \lfloor 2\sqrt{n} \rfloor
$$
证明：
对于$d \leq \lfloor \sqrt{n} \rfloor,\lfloor \frac{n}{d}\rfloor$有$\lfloor \sqrt{n} \rfloor$种取值
对于$d >\lfloor \sqrt{n} \rfloor,有\lfloor \frac{n}{d}\rfloor \leq \lfloor \sqrt{n} \rfloor$ ,也只有$\lfloor \sqrt{n} \rfloor$种取值
得证

3.3数论分块结论：

对于正整数$n$,使得式子$\lfloor\frac{n}{i}\rfloor=\lfloor\frac{n}{j}\rfloor$成立的最大的满足$i\leq j \leq n$的$j$的值为$\lfloor\frac{n}{\lfloor\frac{n}{i}\rfloor}\rfloor$。

证明：
令$k=\lfloor\frac{n}{i}\rfloor$,可以知道
$$k\leq\frac{n}{i}$$
所以
$$\lfloor\frac{n}{k}\rfloor\geq\lfloor\frac{n}{\frac{n}{i}}\rfloor=\lfloor i\rfloor=i$$
所以
$$i\leq j\leq n$$
又由于当
$$\lfloor\frac{n}{i}\rfloor=\lfloor\frac{n}{j}\rfloor$$
有
$$\lfloor\frac{n}{i}\rfloor \leq \frac{n}{j}< \lfloor\frac{n}{i}\rfloor +1$$
所以
$$\frac{1}{\lfloor\frac{n}{i}\rfloor}\geq\frac{j}{n}>\frac{1}{\lfloor\frac{n}{i}\rfloor+1}$$
即
$$\frac{n}{\lfloor\frac{n}{i}\rfloor+1}<j\leq\frac{n}{\lfloor\frac{n}{i}\rfloor}$$
得证$j_{max}=\lfloor\frac{n}{\lfloor\frac{n}{i}\rfloor}\rfloor$
这样便可以在$O(\sqrt{n})$的时间内计算$\sum_{i=1}^{n}\lfloor\frac{n}{i}\rfloor$

4.应用

[SDOI2015]约数个数和

题目描述

设 $d(x)$ 为 $x$ 的约数个数，给定 $n,m$，求
$$\sum_{i=1}^n\sum_{j=1}^md(ij)$$

输入格式

输入文件包含多组测试数据。
第一行，一个整数 $T$，表示测试数据的组数。
接下来的 $T$ 行，每行两个整数 $n,m$。

输出格式

$T$ 行，每行一个整数，表示你所求的答案。

样例 #1

样例输入 #1
2
7 4
5 6
样例输出 #1
110
121

提示

【数据范围】
对于 $100%$ 的数据，$1\le T,n,m \le 50000$。

先将$d(ij)$转换为另一种形式
由于$i=p_1^{\alpha_1}\ldots p_k^{\alpha_k}$,$j=p_1^{\beta_1}\ldots p_k^{\beta_k}$$i*j=p_1^{\alpha_1+\beta_1}\ldots p_k^{\alpha_k+\beta_k}$
所以$d(ij)=(\alpha_1+\beta_1+1)\ldots(\alpha_k+\beta_k+1)$
对于$i,j$如果每次选择的$\alpha\beta$满足$[gcd(\alpha,\beta)=1]$则对每个$p_k$刚好有$\alpha_k+\beta_k+1$种选择
所以
$$d(ij)=\sum_{x|i}\sum_{y|j}[gcd(x,y)=1]$$
则题目所求为
$$\sum_{i=1}^N\sum_{j=1}^M\sum_{x|i}\sum_{y|j}[gcd(x,y)=1]$$
设$$f(n)=\sum_{i=1}^N\sum_{j=1}^M\sum_{x|i}\sum_{y|j}[gcd(x,y)=n]$$
则可设$$F(n)=\sum_{n|d}f(d)=\sum_{i=1}^N\sum_{j=1}^M\sum_{x|i}\sum_{y|j}[n|gcd(x,y)]$$
由$x|i,y|j \Rightarrow 1\leq i\leq N,1\leq j\leq M$
每当$N、M$是$x、y$的倍数时才会使用一次约数$x、y$
故可以将前两个求和与后两个求和进行交换
原式等于
$$\sum_{x=1}^{N}\sum_{y=1}^{M}\lfloor\frac{N}{x} \rfloor\lfloor\frac{M}{y} \rfloor[n|gcd(x,y)]$$
将上式转换为枚举$n$的倍数
即$$\sum_{x^{‘}=1}^{\frac{N}{n}}\sum_{y^{‘}=1}^{\frac{M}{n}}\lfloor\frac{N}{nx{‘}} \rfloor\lfloor\frac{M}{ny{‘}} \rfloor$$
上式使用乘法结合律可以转换为
$$(\sum_{x^{‘}=1}^{\frac{N}{n}}\lfloor\frac{N}{nx{‘}} \rfloor)(\sum_{y^{‘}=1}^{\frac{M}{n}}\lfloor\frac{M}{ny{‘}} \rfloor)$$
令$$h(n)=\sum_{i=1}^n\lfloor\frac{n}{i}\rfloor$$
则$$F(n)=h(\frac{N}{n})h(\frac{M}{n})$$
由莫比乌斯反演(2)式的得
$$f(n)=\sum_{n|d}\mu(\frac{d}{n})F(d)$$
故$$f(1)=\sum_{d=1}^{min(N,M)}\mu(d)F(d)=\sum_{d=1}^{min(N,M)}\mu(d)h(\frac{N}{d})h(\frac{M}{d})$$
上式的预处理h的情况下复杂度在$O(TN)$还是会T
但是观察可以发现如果预处理莫比乌斯函数的前缀和再使用数论分块求$f(1)$就可以降到$O(T\sqrt N)$非常nice

展开代码 


#include<bits/stdc++.h>
using namespace std;

const int N=5e4+5;
int h[N],mobius[N],sum[N];
vector<int>primes;
bool st[N];
void init(int n){
    mobius[1]=1;
    for(int i=2;i<=n;i++){
        if(!st[i]){
            primes.push_back(i);
            mobius[i]=-1;
        }
        for(auto x:primes){
            if(x>n/i) break;
            st[x*i]=true;
            if(i%x==0) break;
            mobius[i*x]=mobius[i]*-1;
        }
    }
    for(int i=1;i<=n;i++) sum[i]=mobius[i]+sum[i-1];
}
int calc(int n){
    int res=0;
    for(int l=1,r;l<=n;l=r+1){
        r=n/(n/l);
        res+=(r-l+1)*(n/l);
    }
    return res;
}
void slove(){
    init(N-1);
    for(int i=1;i<N-1;i++)
        h[i]=calc(i);
    int t;
    cin>>t;
    while(t--){
        int n,m;
        cin>>n>>m;
        if(n>m) swap(n,m);
        long long res=0;
        for(int l=1,r;l<=n;l=r+1){
            r=min(n/(n/l),m/(m/l));
            res+=(sum[r]-sum[l-1])*1ll*h[n/l]*h[m/l];
        }
        cout<<res<<'\n';
    }
}
int main(){
    //freopen("in.txt","r",stdin);freopen("out.txt","w",stdout);
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);std::cout.tie(nullptr);
    int _=1;
    // cin>>_;
    for(int i=1;i<=_;i++){
        slove();
    }
    return 0;
}
